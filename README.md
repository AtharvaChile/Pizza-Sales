# SQL-Pizza-Sales
## 🍕 Pizza Sales Analysis using SQL
### 📌 Project Overview

This project analyzes a pizza sales dataset using SQL to extract meaningful business insights. The goal is to perform data analysis on sales, revenue, customer ordering behavior, and product performance using structured queries.
### 📊 Business Problems Solved

The following key business questions were solved using SQL:

Total number of orders

Total revenue generated

Highest priced pizza

Most common pizza size ordered

Top 5 most ordered pizzas

Total quantity ordered per category

Distribution of orders by hour

Category-wise pizza distribution

Average number of pizzas ordered per day

Percentage contribution of each pizza category to revenue

Top 3 pizzas by revenue

Cumulative revenue over time

Revenue generated per category
